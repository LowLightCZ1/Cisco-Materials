## ___Secure Unused Ports___
- Disable unused interfaces
- Can use __Interface range__ command.
``interface range fa0/*first-num* - *last-num*``
## ___Mitigate MAC Address Table Attacks___
- Enable port security
- It limits the number of valid addresses allowed on port.
- Port receives frame -> source address is compared to the list of secure addresses (manually configured or dynamically learned)
## ___Enable Port Security___
- The __port-security__ command is rejected, why ?
```
interface f0/1
switchport port-security
Command rejected: FastEthernet0/1 is in dynamic port
switchport mode access
switchport port-security
```
- __Port-security__ can be only configured on access (manually config.) or trunk
- Can use __show port-security interface__ command to display the current port settings
## ___Limit and Learn MAC Addresses___
- Default port security = 1
- The maximum num of secure depends on the switch and IOS.
``S1(config-if) switchport port-security maximum *value*``

- The switch can learn about addresses on secure port in one of three ways:
	- __Manually Configured__
		``switchport port-security mac-address *mac-address*``
	- __Dynamically Learned__
		- When __switchport port-security__ is config. the current source address is auto. secured, but not added to the to the startup sett.
	- __Dynamically Learned - Sticky__
		- Dynamically learn addresses and 'stick' them to the running configuration
		``switchport port-security mac-address sticky``
## ___Port Security Aging___
- Can set aging time for static and dynamic _secure address_.
- __Absolute__ - secure addresses on the port are deleted after the specific aging time
- __Inactivity__ -||- only if they are inactive for specified aging time
- Use aging to remove secure MAC without manually deleting the existing MAC.
``switchport port-security aging { static | time *time*  |  type  {absolute | inactivity} ``

| Parameter           | Description                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------------------ |
| ``static``          | Enable aging for statically configured secure addresses on this port                                   |
| ``time *time*``     | Specify the aging time for this port. The range is 0 - 1440 min. If the time is 0, aging is disabled.  |
| ``type absolute``   | Set the absolute aging time. All addresses on this port age out the same time (in minutes).            |
| ``type inactivity`` | Set the inactivity aging type. The addresses age out only if there is no data traffic from the source. |
## ___Port Security Violation Modes___
- If the address of a device differs from the list of secure addresses, a __port violation__ occurs.
- Default, the port enters the _error-disable_ state.
``switchport port-security violation { protect | restrict | shutdown }``

| Mode                   | Description                                                                                                                                               |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ``shutdown`` (default) | Port transition _error-disable_ state immediately, turn of the port LED and sends a syslog message.                                                       |
| ``restrict``           | Port drops packets with unknown source addresses until you remove a sufficient number of secure MAC to drop below maximum value or increse the max. value |
| ``protect``            | The least secure mode. Same as previous mode. No syslog message is sent.                                                                                  |
## ___Verify Port Security__
- __Port security for All Interfaces__
	``show port-security``
- __Port security for Specific Interface__
	``show port-security interface fa0/1``
- __Verify Learned MAC__
	``show run interface fa0/1``
- __Verify Secure MAC__
	``show port-security address``
