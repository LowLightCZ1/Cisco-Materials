## ___DHCP Snooping___
- Doesn't rely on source MAC Addresses
- Instead, determines whether __DHCP messages__ are from an administratively config. __trusted or untrusted__ source. Filters messages and rate-limits traffic.
----
- __Trusted sources__ - switche, routers, servers
- __Untrusted sources__ - everything beyond the firewall or outside your network
---
## ___Steps to Implement DHCP Snooping___
1. Enable DHCP snooping by using ``ip dhcp snooping``.
2. On trusted ports , use ``ip dhcp snooping trust``.
3. Limit the number of DHCP discovery messages that can be received per second using ``ip dhcp snooping limit rate``.
4. Enable DHCP snooping by VLAN, or by range of VLANs, using ``ip dhcp snooping *vlan*``.
