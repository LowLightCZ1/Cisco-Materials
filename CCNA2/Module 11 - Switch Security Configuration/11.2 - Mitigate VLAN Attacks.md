## ___Steps to Mitigate [[10.4 + .5 - MAC Address + LAN Attacks| VLAN Attacks]]___
1. Disable __DTP (auto trunking)__ negotiations on non-trunking ports by using ``switchport mode access``.
2. Disable unused ports and put them in an unused VLAN.
3. Manually enable the trunk link on a trunking port using ``switchport mode trunk``. 
4. Disable __DTP (auto trunking)__ negotiations on trunking ports using ``switchport nonegotiate``.
5. Set the VALN to a VLAN other than VALN 1 by using ``switchport trunk native *vlan_number*``.
