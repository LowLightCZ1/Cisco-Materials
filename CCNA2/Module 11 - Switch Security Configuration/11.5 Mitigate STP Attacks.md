## ___PortFast and BPDU Guard___
- __PortFast__ - immediately brings an interface configured as an access port to the _forwarding state_ from _blocking state_. Apply to all end-user ports.
- __BPDU Guard__ - immediately error-disables a port that receives a BPDU. 

## ___Configure PortFast___
- If PortFast is enabled on a port connecting another switch, can create a spanning-tree loop.
---
- Enable PortFast on an interface - ``spanning-tree portfast``.
- Enable PortFast on all access ports - ``spanning-tree portfast default``.
## ___Configure BPDU Guard___
- Recover from error-disable state - ``errdisable recovery cause bpduguard``
___
- Enable BPDU Guard on an interface  - ``spanning-tree bpduguard enable``
- Enable BPDU Guard on all PortFast-enabled ports - ``spanning-tree portfast bpduguard default``
