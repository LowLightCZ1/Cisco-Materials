## ___Dynamic ARP Inspection___
- Requires __DHCP snooping__ and helps prevent ARP attacks by:
1. Not relaying invalid or gratuitous ARP Request out to other ports in the same VLAN.
2. Intercepting all ARP Requests and Replies on untrusted ports.
3. Verifying each intercepted packet for a valid IP-to-MAC binding.
4. Dropping and logging ARP Requests coming from invalid source.
5. Error-disabling the interface if the config. DAI number of ARP packet is exceeded.
## ___DAI Implementation Guidelines___
1. Enable DHCP snooping globally.
2. Enable DHCP snooping on selected VLANs.
3. Enable DAI on selected VLANs.
4. Configure trusted interfaces for DHCP snooping and ARP inspection.
