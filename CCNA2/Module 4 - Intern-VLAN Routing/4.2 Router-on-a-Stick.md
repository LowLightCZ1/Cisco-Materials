## ___S1 VLAN and Trunking Configuration___
 ___Steps:___
1. Create and name a VLANs
	```
	S1(config)# vlan 10
	S1(config-vlan)# name LAN10
	S1(config-vlan)#exit
	-||-
	```
2. Create and management interface
	```
	S1(config)# interface vlan99
	S1(config-if)# ip add 192.168.99.2 255.255.255.0
	S1(config-if)# no shut
	S1(config-if)# exit
	S1(config)# ip default-gateway 192.168.99.1
	```
3. Configure access port
	```
	S1(config)# interface fa0/6
	S1(config-if)# switchport mode access
	S1(config-if)# switchport access vlan10
	S1(config-if)# no shut
	S1(config-if)# exit
	```
4. Configure trunking port
	```
	S1(config)# interface fa0/1
	S1(config-if)# switchport mode trunk
	S1(config-if)# no shut
	S1(config-if)# exit
	S1(config)# interface fa0/5
	S1(config-if)# switchport mode trunk
	S1(config-if)# no shut
	S1(config-if)# exit
	```

## ___R1 Subinterface Configuration___
- Subinterface must be assigned an __IP__ or __Unique subnet__

- Create subinterface
	``interface "interface_id.subinterface_id"``

- Subinterface to respond __802.1Q encapsulated traffic__
	``encapsulation dot1q "vlan_id"[native]``

- Configures __IPv4__ address of the subinterface (serves as Default Gateway)
	``ip address "ip-address subnet-mask" ``

- Enable physical interface
	``no shutdown``
