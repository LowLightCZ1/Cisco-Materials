| Issue Type            | How to fix                                                                                    |
| :-------------------- | --------------------------------------------------------------------------------------------- |
| Missing VLAN          | - Create VLAN if not exist<br>- Ensure host port is assigned to correct VLAN                  |
| Trunk Port Issue      | - Ensure ports are assigned correctly<br>- Ensure port is trunk and enable                    |
| Access Port Issue     | - Assigned correct VLAN<br>- Ensure port is access and enable                                 |
| Router Config. Issues | - Router subinterface is assigned to the VLAN ID<br>- Router subinterface is configured wrong |

## ___Missing VLAN___
- Not created
- Accidently deleted
- Not allowed on trunk link

- VLAN10 is missing and port Fa0/6 is not assigned to default VLAN. 
- Use ``show interface 'interface-id switchport'
```
S1(config)# do show interface fa0/6 switchport
Name: Fa0/6
Switchport: Enabled
Administrative Mode: static access
Administrative Trunkig Encapsulation: dot1q
Operational Trunkig Encapsulation: native
Negotiation of Trunking: Off
--Access Mode VLAN: 10 (Iactive)--
```
- __Solution:__ Recreate VLAN10
