- L2 connection between switches that carries all VLAN traffic
- Use 802.1Q frame tagging

## ___Trunk Configuration Commands___
__Example:__
- VLAN 10 - Faculty - 172.17.10.0
- VLAN 20 - Students - 172.17.20.0
- VLAN 30 - Guests - 172.17.30.0
- VLAN 99 - Native - 172.17.99.0
```
S1(config)# interface fastEthernet 0/1
S1(config-if)# switchport mode trunk 
S1(config-if)# switchport trunk native vlan 99
S1(config-if)# switchport trunk allowed vlan 10 20 30 99
S1(config-if)# end
```

## ___Verify Trunk Configuration___
``show interface f0/1 switchport``
``show interface trunk``
## ___Reset the Trunk___
```
S1(config)# interface f0/1
S1(config-if)# no switchport truk allowed vlan
S1(config-if)# no switchport truk native vlan
S1(config-if)# end
```
