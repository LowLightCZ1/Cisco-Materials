## ___Negotiated Interface Modes___
```
S1(config-if)# switchport mode { access | dynamic (auto | desirable) | trunk }
```
- __access__ - permanent nontrunking mode  
- __dynamic auto__ - able to convert the link to a trunk link
- __dynamic desirable__ - actively attempt to convert the link to a trunk link 
- __trunk__ - permanent trunking mode
