___Commands:___
- ``show interface port-channel *identifier*`` - General status of port-channel interface
- ``show etherchannel summary`` - Display one line of information per port channel.(Flags)
- ``show etherchannel port-channel`` - Display information about a specific port channel interface.
- ``show interface *interface* etherchannel`` - Provide info. about the role of the interface in the EtherChannel.

## ___Common Issue with Config.___
- All interfaces must have the same 
	- __speed and duplex mode__.
	- __native and allowed VLAN on trunks__.
	- __access VLAN on access port__.
- ___Issues:___
	- Ports are not part of the same VLAN ( or not configure as trunk)
	- Trunking was configured on some, but not all ports to make up EtherChannel.
	- Different allowed range of VLAN.
	- PAgP and LACP are not compatibly config. on both ends of EtherChannel.

```
		 M - not in use, minimum links not met

		 w - waiting to be aggregated

		 A - formed by Auto LAG

Group  Port-channel  Protocol    Ports
------+-------------+-----------+------------------------------
1       Po1(SD)       -        Fa0/1(D)    Fa0/2(D)
```
- __Step 2:__  View Port Channel config.
```
S1# show run | begin interface port-channel
```
- __Step 3:__ Correct the Misconfiguration
```
S1(config)# no interface port-channel 1
S1(config)# interface range fa0/1 - 2
S1(config-if-range)# channel-group 1 mode desirable 
S1(config-if-range)# no shutdown 
S1(config-if-range)# exit
S1(config)# interface port-channel 1
S1(config-if)# switchport mode trunk 
S1(config-if)# end
S1#
```
