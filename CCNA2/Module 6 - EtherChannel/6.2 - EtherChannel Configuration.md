___Restrictions:___
- __EtherChannel support__ - Interfaces must support EtherChannel.
- __Speed and Duplex__ - Must operate at the same speed and in the duplex mode.
- __VLAN Mach__ - Interfaces in bundle must be assigned into the same VLAN (or be trunk).
- __Range of VLANs__ - Supports the same allowed range of VLANs on all interfaces

Changes must be configured in __port channel interface configuration mode__ (PICM)

## ___LACP Config. Example___
 ![[Screenshot 2026-01-08 191433.png]]
 - EtherChannel is disabled.
 - ___Steps:___
	 1. Specify the interface in the EtherChannel group ``interface range *interface*``
	 2. Create the port channel interface ``channle-group *identifier* mode active``
	 3. Change settings on the channel port interface ``interface port-channel *identifier*``
```
S1(config)# interface range FastEthernet 0/1 - 2
S1(config-if-range)# chnnel-group 1 mode active
S1(config-if-range)# exit
S1(config)# interface port-channel 1
S1(config-if)# switchmode port trunk
S1(config-if)# switchmode trunk allowed vlan 1,2,20  
```
