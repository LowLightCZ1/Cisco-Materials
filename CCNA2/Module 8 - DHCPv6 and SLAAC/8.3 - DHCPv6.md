- Defined in __RFC 3315__
- ___Steps in Communication__:
	1. __Host sends an RS message__
	2. __Router responds with an RA message__
	3. __Host sends a DHCPv6 SOLICIT message__
	4. __The DHCPv6 responds with an ADVERTISE message__ 
	5. __Host responds to the DHCPv6 server__
	6. __DHCPv6 send a REPLY message__
## ___Stateless DHCPv6 Operation___
- Only providing information that is identical for all devices (IPv6 address)
- Server is not maintaining any client state info.
- Only providing config. parameters for client
