## ___RA Flags___
- **A Flag** - Address Autoconfig. Use SLAAC to create IPv6 GUA
- **O Flag** - Other Config. Other info. available from stateless DHCPv6 server 
- **M Flag** - Managed Address Config. Use a stateful DHCPv6 to obtain a GUA
