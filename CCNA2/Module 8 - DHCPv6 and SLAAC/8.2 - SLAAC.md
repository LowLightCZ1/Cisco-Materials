- Enables to create their own unique IPv6 [[8.1 - IPv6 GUA Assignment |GAU]] without DHCPv6 server.
- Stateless service - no server thet maintains network address info. 
- Uses **RA** messages to provide addressing (normally provided by server).
	- Sends every 200 sec.7
	- __SLAAC Only__ - A=1, O=0, M=0
	- __Stateless DHCPv6__ - A=1, O=1, M=0
	- __DHCPv6__ - A=0, M=1

## ___Enabling SLAAC___

```
R1# show ipv6 interface G0/0/1
```
- __Enable IPv6 Routing__
```
R1(config)# ipv6 unicast-routing
R1(config)# exit
```
- __Verify SLAAC__
```
R1# show ipv6 interface G0/0/1 | section Joined
```

## ___SLAAC Only Method___
- Enabled by default
- Enabled __IF(interface)__ with GUA start sending RA messages with __Flags__
	- A = 1 - Suggest to the client own GUA using prefix advertised in the RA
	- O = 0 - instruct to use the info. in RA exclusively(prefix, prefix-length, DNS server, MTU)
	- M = 0 - Same as __O flag__
- ![[Screenshot 2026-02-16 132553.png]]
## ___ICMPv6 RS Messages___
- When client configured to obtain info. automatically, it sends __RS__ to __ff02::2__.
## ___Generate IF(Interface) ID___
- 64-bit ID
- __Randomly generated__
	- by client operating system
	- used by Windows 10 hosts
- __EUI-64_-
	- using MAC address
	- Hex value of __FFFE__ in the middle of address
	- flips the seventh bit
