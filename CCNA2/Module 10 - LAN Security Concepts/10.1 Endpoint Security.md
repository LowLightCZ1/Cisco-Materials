## ___Network Attacks___
- __Distributed Denial of Service(DDoS)__
	- Coordinated attack from many devices. 
	- Degrading/halting public access to an organization website.
- __Data Breach__
	- Organization data server or host are compromised to steal confidential info.
- __Malware__

## ___Network Security Devices___
- __VPN-Enabled Router__
	- Provides a secure connection to remote users.
	- Can be integrated into Firewall
- __NGFW(Next-Generation Firewall)__
	- Provides stateful packet inspection, ...
		- URL filtering 
		- advance malware protection(AMP)
- __NAC(Network Access Control)__
	- Includes __AAA__ - authorization, authentication, accounting

## ___Cisco Email + Web Security Appliance___
### ___Email Security Appliance___
- Designed to monitor __SMTP__(Simple Mail Transfer Protocol)
- Constantly updated in real-time feeds from Cisco Talos
- data pulled every __three to five minutes__
- ___Functions of Cisco ESA___
	1. Blocks known threats.
	2. Discard emails with bad links.
	3. Blocks access to newly infected sites.
	4. Remediate against stealth malware.

### ___Web Security Appliance___
- Helps organizations address the challenges of securing and controlling web traffic.
- Complete control over how users access the internet.
- ___Functions of Cisco WSA___
	- Blacklisting of URL
	- URL filtering
	- Malware scanning
	- URL categorization
