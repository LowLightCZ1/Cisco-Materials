## ___[[10.1 Endpoint Security|AAA]] Components___
- Similar to using a credit card
### __Authentication__
Two methods
- __Local Authentication__
	- Stores data in the local network device
	- Users authenticate against local database
- __Server-based Authentication__
	- Router access central AAA servers
	- Router uses either _RADIUS_ or _TACACS+_ to communicate
> Note: _RADIUS (Remote Authentication Dial-In Users Service)_; _TACACS+(Terminal Access Controller Access Control System)_
### __Authorization__
- After authentication, authorization is automatic
- governs what users can or can't do on network
### __Accounting__
- Collect and report usage data.
- Data -> number of frames, number of packets, executed commands
## ___IEEE 802.1X___
- Port-based access control protocol
- Restricts unauthorized workstations from connecting to a LAN
- ___Roles:___
	- __Supplicant(Client)__ 
	- __Authenticator(Switch)__ 
	- __Authentication server__
