- __Layer 2 Attacks__

| Category                     | Examples                                                                 |
| ---------------------------- | ------------------------------------------------------------------------ |
| __MAC Table Attacks__        | Includes MAC address flooding attacks.                                   |
| __VLAN Attacks__             | Includes VLAN hooping and also attacks between devices on a common VLAN. |
| __DHCP Attacks__             | Includes DHCP starvation / DHCP spoofing attacks.                        |
| __ARP Attacks__              | Includes ARP spoofing / ARP poisoning attacks.                           |
| __Address Spoofing Attacks__ | Includes MAC + IP addresses spoofing attacks.                            |
| __STP Attacks__              | Includes Spanning Tree Protocol manipulation attacks.                    |
- __Layer 2 Attacks Mitigation__

| Solution                         | Description                                                             |
| -------------------------------- | ----------------------------------------------------------------------- |
| __Port Security__                | Prevent many types of attacks including MAC flooding / DHCP starvation. |
| __DHCP Snooping__                | Prevent DHCP starvation / DHCP spoofing.                                |
| __DAI (Dynamic ARP Inspection)__ | Prevent ARP spoofing / ARP poisoning.                                   |
| __IPSG (IP Source Guard)__       | MAC + IP address spoofing.                                              |
- Always use secure types of protocols 
- Use out-of-band management network
- Use a dedicated management VLAN 
- Use ACLs to filtre unwanted access
