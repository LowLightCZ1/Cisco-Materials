Attacks can be generated by __outsiders__, __disgruntled employees__ or by mistake by employees.
___
- __Interception of Data__ - Wireless data should be encrypted to prevent it from being read.
- __Wireless Intruders__ - Unauthorized users attempting to access network recourses.
- __DoS (Denial of Service)__ 
- __Rouge APs__ - Unauthorized APs installed by a well-intentioned user or for malicious purposes. Can be detected by using management software.
___
## ___DoS Attacks___
Can be the result of:
- __Improperly configured devices__ 
- __A malicious user intentionally interfering with the wireless communication__
- __Accidental interfaces__
How to minimize the risk:
- __Harden all devices__
- __Keep password secure__
- __Create backups___
- __Check that all configuration are incorporated off-hours__
---
## ___Rouge APs Attacks___
- AP or wireless router connected without authentication and against corporate policy.
- Can be used by an attacker to capture __MAC addresses__, __data packets__, __gain access to network resources__.
How to prevent it:
- Configure WLCs with rouge AP policies
- Use monitoring software.
___
## ___Man-in-the-Middle Attack___
- The hacker is positioned in between two legitimate entities.
- Want to __read__ or __modify__ the data that passes between the two parties.
Ways to create MITM:
- __"evil twin AP"__ attack - introduces rouge AP, configured with same SSID as a legitimate AP.
