## ___SSID Cloaking + MAC Filtering___
___SSID Cloaking___
- Wireless clients must manually configure the SSID to connect to the network.
___MAC Address Filtering___
- An administrator can manually permit or deny clients wireless access based on their MAC address
___
## ___802.11 Original Authentication Methods___
- __Open system authentication__
	- Any client should easily be able to connect and should only be used in situations where security is no concern (hotels, cafes, remote areas).
	- Client is responsible for providing security.
- __Shared key authentication__ - Provides mechanisms (WEP, WPA, WPA2/3) to authenticate and encrypt data between clients and APs.
- The password must be pre-shared.
___
## ___Shared Key Authentication Methods___

|       Authentication Methods       | Description                                                                                                                                                               |
| :--------------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Wired Equivalent Privacy (WEP)** | Original 802.11 specification. Using the Rivest Cipher (RC4). Static key. No longer used.                                                                                 |
|  **Wi-Fi Protected Access (WPA)**  | Uses WEP, but secure data with Temporal Key Integrity Protocol (TKIP) algorithm. Changes key for each packet.                                                             |
|              **WPA2**              | Current industry standard. Uses Advanced Encryption Standard (AES).                                                                                                       |
|              **WPA3**              | The next generation of Wi-Fi security. All WPA3-enabled devices use the latest security methods. Require the use of Protected Management Frames (PMF). Not yet available. |
___
## ___Authenticating a Home User___
- __Personal Method__ 
	- Users authenticate using pre-shared key (PSK). 
	- Wireless clients authenticate with pre-shared password.
- __Enterprise Method__ 
	- Requires __Remote Authentication Dial-In User Server (RADIUS)__ server.
	- Provides additional security.
	- device must be authenticated by the RADIUS server.
___
## ___Encryption Methods___
- __Temporal Key Integrity Protocol (TKIP)__
	- Used by WPA
	- Support for legacy WLAN equipment.
	- Encrypts Layer 2 payload using TKIP, and __MIC (Message Integrity Check)__.
- __Advanced Encryption Standard (AES)__
	- Used by WPA2
	- Uses Counter Cipher Mode with __Block Chaining Message Authentication Code Protocol (CCMP)__.
___
## ___Authentication in the Enterprise___
- Enterprise security mode requires AAA RADIUS server.
- __RADIUS Server IP__ - reachable address of the RADIUS server.
- __UPD port numbers__ 
- __Shared Key__ - Used to authenticate the AP with the RADIUS server.
