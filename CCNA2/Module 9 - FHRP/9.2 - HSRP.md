- Cisco-proprietary FHRP that allow transparent failover of FHIP(first-hop IP) device
- Hight network availabitily
- Provided for IP hosts with default gateway
- used in groups of routers for selecting a active and standby device
## ___Priority and Preemption___
- Numerically highest IPv4 = active router
- __Priority__
	- highest priority = active router
	- default - 100
	- standby priority - 0 to 255
- __Preemption__
	- ability of an HSRP to trigger the re-election process.
	- use __standby preempt__.

## ___States and Timers___
- __Active HSRP__
	- forwarding traffic for the segment
- __Passive HSRP__
	- standby, ready to assume the active role (when fail)
- When configured, router __sends and receives__ hello packets 
- Starts determining which states it will assume in the group


| HSRP State | Description                                                                                                                                 |
| :--------: | :------------------------------------------------------------------------------------------------------------------------------------------ |
|  Initial   | entered through a configuration change or an interface become available                                                                     |
|   Learn    | router has not determine virtual IP address + has not yet seen __hello__ message from a active router. Waits to hear from the active router |
|   Listen   | router knows virtual IP, but is neither the active router nor the standby router. Listens for __hello__ messages from those routers         |
|   Speak    | router sends hello messages and actively participate in the election                                                                        |
|  Standby   | router is a candidate to become the next active router and sends hello messages                                                             |
|   Active   | router won the election                                                                                                                     |
