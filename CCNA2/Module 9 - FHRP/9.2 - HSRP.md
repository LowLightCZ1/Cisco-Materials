- Provided for IP hosts with default gateway

## ___Priority and Preemption___
- Numerically highest IPv4 = active router
- __Priority__
	- highest priority = active router
	- default - 100
	- standby priority - 0 to 255
- __Preemption__
	- ability of an HSRP to trigger the re-election process.
