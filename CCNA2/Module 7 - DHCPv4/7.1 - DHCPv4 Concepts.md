- Assigns IPv4 addresses and others network config. info. dynamically.
- Small branches or SOHO - DHCPv4 can provide service without dedicated server.
- Dynamically assigns IPv4 address from pool of addresses of limited period of time.

## ___DHCPv4 Operations___
- Works in __client/server__ mode.
- Client must ask periodically for extend the lease.

## ___DHCPv4 Steps___
- __To obtain a lease__
	- DHCP Discovery - Client broadcast message (MAC address) to discover available server.
	- DHCP Offer - Reserve IPv4 address to a lease. Create ARP entry (MAC client, IP lease).
	- DHCP Request - Used for _lease origination_ and _lease renewal_. 
	- DHCP Acknowledgment - Verify info. with ICMP ping, create new ARP entry, duplicate DHCPOFFER but different field type.
- __To renew a lease__
	- DHCP Request 
	- DHCP Acknowledgment
