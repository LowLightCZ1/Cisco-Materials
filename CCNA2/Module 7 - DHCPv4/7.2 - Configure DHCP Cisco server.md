- Router can serve as DHCP server
___Steps:___
1. Exclude IPv4 addresses.
	- Need to specific addresses
	- Some IPv4 are assigned to network device.
	``Router(config)# ip dhcp excluded-address *low-address* [hight-adddress]``
2. Define a DHCPv4 pool name.
	```
	  Router(config)# ip dhcp pool *pool-name*
	  Router(dhcp-config)#
	```
3. Configure a DHCPv4 pool.
	- Address pool and __default gateway__ must configured.
	- __Network__ - range and available addresses
	- __Default-router__ - define default gateway, can set up to eight addresses
	- __dns-server__ - available DNS server
	- __lease__ - config. duration of a lease
	- __netbios-name-server__ - define NetBIOS WINS server

|            **Task**            |                       IOS Command                       |
| :----------------------------: | :-----------------------------------------------------: |
|    Define the address pool     | ``network *network-number* [mask \| / prefix-length]``  |
|   Define the default gateway   |   ``default-router *address* [address2...address8]``    |
|      Define a DNS server       |     ``dns-server *address* [address2...address8]``      |
|     Define the domain name     |                 ``domain-name *name*``                  |
| Define the duration of a lease |       ``lease {days[hours[minutes] \| infinite}``       |
| Define the NetBIOS WINS server | ``netbios-name-server *address* [address2...address8]`` |

___Example:___
![[Screenshot 2026-01-29 115807.png]]
```
R1(config)# ip dhcp excluded-address 192.168.10.1 192.168.10.9
R1(config)# ip dhcp excluded-address 192.168.10.254
R1(config)# ip dhcp pool LAN-POOL-1
R1(dhcp-config)# network 192.168.10.1 255.255.255.0
R1(dhcp-config)# default-router 192.168.10.1
R1(dhcp-config)# dns-server 192.168.11.5
R1(dhcp-config)# domain-name example.com
R1(dhcp-config)# end
R1#
```

## ___Verification Commands___
``show running-config | section dhcp`` - Display commands configurate on the router
``show ip dhcp binding`` - Display a lost of IPv4 addresses to MAC addresses bindings provided by server
``show ip dhcp server statistic`` - Display count information regarding the num. of DHCPv4 messages that have been sent.
