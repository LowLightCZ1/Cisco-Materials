## ___Secure Unused Ports___
- Disable unused interfaces
- Can use __Interface range__ command.
``interface range fa0/*first-num* - *last-num*``
## ___Mitigate MAC Address Table Attacks___
- Enable port security
- It limits the number of valid addresses allowed on port.
- Port receives frame -> source address is compared to the list of secure addresses (manually configured or dynamically learned)
## ___Enable Port Security___
- The __port-security__ command is rejected, why ?
```
interface f0/1
switchport port-security
Command rejected: FastEthernet0/1 is in dynamic port
switchport mode access
switchport port-security
```
- __Port-security__ can be only configured on access (manually config.) or trunk
- Can use __show port-security interface__ command to display the current port settings
## ___Limit and Learn MAC Addresses___
- Default port security = 1
- The maximum num of secure depends on the switch and IOS.
``S1(config-if) switchport port-security maximum *value*``

- The switch can learn about addresses on secure port in one of three ways:
	- __Manually Configured__
		``switchport port-security mac-address *mac-address*``
	- __Dynamically Learned__
		- when __switchport port-security__ is config. the current source address is auto. secured, but not added to the to the startup sett.
	- __Dynamically Learned - Sticky__
		- Dynamically learn addresses and 'stick' them to the running configuration
		``switchport port-security mac-address sticky``
## ___Port Security Aging___
- __Absolute__ - secure addresses on the port are deleted after the specific aging time
- __Inactivity__ -||- only if they are inactive for specified aging time
